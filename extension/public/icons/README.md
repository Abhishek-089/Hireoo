# Extension Icons

This directory should contain the following icon files:

## Required Icons

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Icon Generation

You can generate these icons from a single SVG or high-resolution PNG using online tools:

1. **Favicon.io**: https://favicon.io/favicon-converter/
2. **RealFaviconGenerator**: https://realfavicongenerator.net/
3. **Favicon Generator**: https://www.favicon-generator.org/

## Icon Design Guidelines

- Use the Hireoo brand colors (#2563eb for blue, #ffffff for white)
- Simple, recognizable design
- Scalable SVG format recommended for generation
- Ensure good contrast for visibility in Chrome toolbar

## Placeholder

For development, you can use any 16x16, 32x32, 48x48, and 128x128 PNG files.
The extension will load but may show default Chrome extension icon.
